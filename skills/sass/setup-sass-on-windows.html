To setup SASS on Windows:

  1. 1. install Ruby https://rubyinstaller.org/
  2. 2. Run cmd and enter: gem install sass
Compile input.scss to output.css:
	sass input.scss output.css
	
To keep SASS compiled to CSS whenever it's changed:
	sass --watch input.scss:output.css
	
To keep all SASS in one dir compiled to CSS in another dir:
	sass --watch app/sass:public/stylesheets

To see full documentation:
	sass --help
